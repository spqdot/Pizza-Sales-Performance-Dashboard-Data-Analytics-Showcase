
🍕 Pizza Sales Performance Dashboard | Data Analytics Showcase
📌 Project Overview

The Pizza Sales Performance Dashboard is a business intelligence project developed to analyze sales performance, product trends, and customer purchasing behavior.

Built using Microsoft Power BI, this dashboard transforms raw transactional sales data into actionable insights that support strategic decision-making in marketing, operations, pricing, and inventory management.

The objective of this project is to demonstrate how data visualization and analytical modeling can uncover revenue drivers and performance patterns within a retail food business environment.
